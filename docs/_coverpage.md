# NPS <small>0.26.10</small>

> A lightweight, high-performance and powerful intranet penetration proxy server

- Supports almost all common protocols
- Supports intranet HTTP proxy, intranet SOCKS5 proxy, P2P and more
- Simple yet powerful web management UI
- Server-side and client-side control
- Powerful extension features
- Cross-platform, one-click installation as a system service
